# SPEC

## MVP 功能

### 1. 學生管理

- 新增學生
- 刪除學生
- 檢視學生清單
- 使用勾選框選取學生

### 2. 活動模板管理

- 新增常用活動模板
- 刪除活動模板
- 內容包含：活動名稱、分類、預設時數、備註
- 這些模板只作為下拉選單使用，主要目的是自動帶入欄位

### 3. 志工時數紀錄

- 選擇多位學生
- 選擇活動模板
- 輸入時數或時長
- 一次建立多筆紀錄

### 4. 資料總覽頁

- 顯示所有學生的一列一筆資料
- 每筆資料顯示學生名稱、年級（由 age 推算）、總時數或指定類別時數
- 支援依年級或時數排序
- 可作為簡易查詢與統計頁面

## 非目標

- 複雜權限管理
- 多角色系統
- 高度自訂流程
- 進階統計報表
